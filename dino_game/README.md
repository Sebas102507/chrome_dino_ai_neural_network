### Instalation

run : 
pip install pygame 

or 


pip3 install pygame


### Running:


run:

python3 chromedino.py


### Input:

| Keys              | Actions                                                       |
|-------------------|---------------------------------------------------------------|
|  `Any Key`        |    Press any key to Start the Game.                           | 
|   **&#8593;**     |    Press `Up Arrow` to jump and avoid cacti.                  |
|   **&#8595;**     |    Press `DOwn Arrow` to duck and avoid pterodactyls.         |
